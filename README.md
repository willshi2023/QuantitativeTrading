程序可以基于已设定的业务逻辑，进行虚拟货币价格的监控、买入卖出计算、盈亏分析。提供多种途径的在家+远程提醒功能，并涵盖健康呵护等功能。它是免费的、可靠的散户全自动量化交易解决方案。

1. **MA上交买入**
2. **MA下交卖出**

4. **诱多陷阱逃离**
